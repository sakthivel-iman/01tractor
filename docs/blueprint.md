# **App Name**: Tractor Status

## Core Features:

- Login Page: User authentication and authorization using a secure login page.
- Speedometer Display: Display the tractor's current speed in kmph using a speedometer-style visual representation.
- Wheel Slip Indicators: Show real-time wheel slip percentages for both rear wheels (left and right).
- Working Hours Chart: Present the number of hours the tractor has been working via bar charts.
- Geolocation Tracking: Track and display the tractor's current location on a map interface similar to Google Maps.

## Style Guidelines:

- Primary color: Dark blue (#1E3A8A), evoking a sense of trust and reliability.
- Background color: Very light blue (#F0F9FF) creates a calm and clean base.
- Accent color: Bright yellow (#D97706) to highlight critical data and interactive elements, providing contrast and drawing the user's attention.
- Font pairing: 'Space Grotesk' (sans-serif) for headlines, 'Inter' (sans-serif) for body text.
- Use clean, modern icons to represent tractor status, wheel slip, and other metrics.
- Dashboard-style layout with clear sections for each data type (speed, wheel slip, working hours, location).
- Subtle animations for updating data displays (e.g., speedometer needle movement, chart updates).
